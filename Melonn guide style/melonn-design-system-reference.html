Instrucción para Claude Code

Pega este bloque al inicio de cualquier sesión con Claude Code para garantizar identidad visual Melonn.

Versión corta
# Design System Melonn Usa melonn-design-system.css como única fuente de estilos. Importa el archivo y usa solo variables CSS y clases definidas. Colores: var(--melonn-*) y var(--color-*). Nunca hex hardcoded. Tipografía: Sora · Tema: light/dark via [data-theme="dark"] · Acento: #4D2AAD.
Versión completa (recomendada)
## Design System Melonn — Reglas de uso Archivo: melonn-design-system.css Importar: <link rel="stylesheet" href="/melonn-design-system.css"> Google Font: Sora (400, 500, 600, 700) ### Reglas obligatorias 1. Colores: SOLO var(--melonn-*) y var(--color-*). Nunca hex/rgb hardcoded. 2. Tipografía: var(--font-primary) = Sora. Tamaños: var(--text-*). 3. Espaciado: SOLO var(--space-*). Nunca px hardcoded. 4. Radios: var(--radius-*). Sombras: var(--shadow-*). 5. Componentes: usar clases .btn .card .stat-card .badge .alert .form-input .sidebar .topbar ### Dark Mode - Activar: document.documentElement.setAttribute('data-theme','dark') - Desactivar: document.documentElement.removeAttribute('data-theme') - El archivo CSS ya incluye el bloque [data-theme="dark"] con todos los tokens overrideados. - Fondo dark: --color-bg: #0A0812 · Surface: --color-surface: #100E1E - Textos dark: --color-text-primary: #EAE6FF · secundario: #8B80B8 ### Identidad Melonn - Light: fondo --color-bg: #F4F5F7 / Dark: fondo --color-bg: #0A0812 - Primario: --melonn-purple: #4D2AAD — botones, sidebar activo - Oscuro: --melonn-navy: #14095B — textos, gradientes - Acento 1: --melonn-teal: #00DCD6 - Acento 2: --melonn-yellow: #FFC800 — CTAs destacados - Primera stat card: siempre .stat-card-featured (gradiente purple→navy) ### Prohibido - No usar frameworks CSS externos (Tailwind, Bootstrap, etc.) - No usar Google Fonts distintos a Sora - No hardcodear colores, spacing ni font-sizes

Logos

Assets oficiales Melonn. Usar siempre el SVG inline o como archivo — nunca redimensionar con distorsión ni cambiar colores manualmente.

Logotipo negro — sobre fondo claro
Logotipo_negro.svg · viewBox 814.51×167.21 · fill #000
Logotipo blanco — sobre fondo oscuro o de color
Logotipo_blanco.svg · viewBox 814.51×167.21 · fill #fff
Isotipo negro — sobre fondo claro
Isotipo_negro.svg · viewBox 379.85×243.02 · fill #000
Isotipo blanco — sobre fondo oscuro o de color
Isotipo_blanco.svg · viewBox 379.85×243.02 · fill #fff
Uso correcto
✅ Logotipo negro — fondos blancos o claros (#F4F5F7, #FFFFFF)
✅ Logotipo blanco — fondos oscuros o de color (navy, purple, dark mode)
✅ Isotipo — cuando el espacio es reducido o cuadrado (favicons, avatares, badges)
❌ No cambiar los colores del SVG — usar solo las versiones negro o blanco
❌ No distorsionar proporciones — siempre escalar con width o height, nunca ambos fijos
